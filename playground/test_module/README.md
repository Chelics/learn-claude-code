# test_module

A simple Python module for demonstration purposes.

## Description

This module contains a basic function `hello()` that prints a greeting message to the console.

## Usage

To use the hello function:

```python
from test_module.main import hello

hello()  # Prints: Hello from test_module!
```

## Installation

No installation required. Simply import the module in your Python script:

```python
import test_module.main
test_module.main.hello()
```
