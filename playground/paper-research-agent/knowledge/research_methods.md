# 计算机科学研究方法论

## 研究类型识别

### 1. 理论研究 (Theoretical Research)
- **特征**: 数学证明、算法分析、复杂度理论
- **关键词**: prove, analysis, complexity, bound, theorem
- **评估标准**: 理论完整性、证明准确性

### 2. 实证研究 (Empirical Research)
- **特征**: 实验验证、性能测试、对比分析
- **关键词**: experiment, evaluation, benchmark, results, performance
- **评估标准**: 实验设计、数据充分性、可复现性

### 3. 系统研究 (Systems Research)
- **特征**: 原型系统、架构设计、实现细节
- **关键词**: system, architecture, implementation, design, prototype
- **评估标准**: 实用性、可扩展性、创新性

### 4. 综述研究 (Survey Research)
- **特征**: 文献梳理、趋势分析、分类总结
- **关键词**: survey, review, overview, taxonomy, literature
- **评估标准**: 覆盖广度、分类合理性、见解深度

## 论文质量指标

### 高质量论文特征
1. **清晰的问题陈述**
2. **完整的实验设计**
3. **充分的对比分析**
4. **详实的结果讨论**
5. **合理的结论推导**

### 创新性评估维度
1. **理论创新**: 新理论、新模型、新算法
2. **技术创新**: 新方法、新架构、新系统
3. **应用创新**: 新应用场景、新问题域
4. **性能提升**: 显著的性能改进

## 关键词优化策略

### AI/ML 领域关键词
- **基础**: machine learning, deep learning, neural networks
- **架构**: transformer, attention mechanism, convolution
- **任务**: classification, generation, optimization
- **方法**: supervised learning, unsupervised learning, reinforcement learning

### 系统领域关键词
- **基础**: distributed systems, parallel computing, cloud computing
- **架构**: microservices, serverless, containerization
- **性能**: scalability, reliability, availability
- **方法**: fault tolerance, load balancing, consistency
